installation of django web frame work in ubuntu : 

1) Create a new Directory and cd into it.

mkdir test , cd test

2) Install and Create a Virtual environment.

python3 -m pip install virtualenv virtualenv venv -p python3

3) Activate Virtual Environment:  source venv/bin/activate

4) Install Django: pip install django

5) Start a new project:â€Šdjango-admin startproject myproject

cd to your project and Run Project:

cd myproject,

6) python3 manage.py runserver
You can see your project here: http://127.0.0.1:8000/  so this our localhost address or port address 

7) now go to the chrome url and type : http://127.0.0.1:8000/ then django default page will be opened......

from django.shortcuts import render
from django.http import HttpResponse  # HttpResponse is belongs to http module   

# requse made by client to get the home page
def home(request):#  path('',views.home,name='home')  path '' means chrome url is changed to /
    # views.home is a func. and the function name is home
   return render(request,'home.html',{'name':'anil'}) 



def add(request) : #  path('add',views.add,name='add') path 'add' means chrome url is changed to /add?num1=1&num2=3
    val1 = int(request.GET['num1']) # actually url data in the form of string so convert it into integer 
    val2 = int(request.GET['num2']) # num2 , num1 names already given to the input fields in home.html file 
    res = val1+val2

    return render(request,'result.html',{'result':res})
